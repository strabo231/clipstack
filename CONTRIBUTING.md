# Contributing to ClipStack

Thanks for your interest in improving ClipStack!

## Reporting Bugs
- Include OS and clipboard tool (xclip/pbcopy)
- Provide command and error
- Steps to reproduce

## Suggesting Features
- Explain the use case
- Consider cross-platform compatibility
- Provide examples

## Pull Requests
1. Fork repository
2. Create feature branch
3. Test on Linux and macOS if possible
4. Follow code style
5. Update documentation

## Testing
Test with sample clipboard data, not sensitive information.
